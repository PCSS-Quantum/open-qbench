Quantum Benchmarks Docs
=====================

----------------
About project
----------------
The main goal of this library is providing various ways to test different quantum computers.

----------------
Getting started
----------------
For now the library is not hosted on pypi so to install run

::

   pip install .

or with dependencies

::

   pip install .[VQE,QSVM,IBM,AQT,ORCA]

Then you can get to know the library by checking out the following sections:

   :maxdepth: 1

   examples
   tutorials
   API Documentation <API/open_qbench>
